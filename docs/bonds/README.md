# Lorebound portraits

Drop a portrait here named after the lorebound and the catalog shows it automatically.
No code edits needed. The letter placeholder stays until a matching file exists.

Naming: the lorebound name, minus the leading "The", lowercased, spaces to hyphens.
Tried in order: .png .jpg .jpeg .webp

Expected filenames:
aerostrix, alkagoo, aquafin, boreal, cawmarch, drakelith, felionis, grimgrit,
hexarach, lumitoad, mordel, runesteed, slipfang, solmera, thistlewing, vixel,
worgar, xenophis

Override: to point one entry at an outside URL (a Wix media link, for example),
set its `image` field in bondforge.html and that wins over this folder.
